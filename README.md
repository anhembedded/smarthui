# Hướng dẫn chạy ứng dụng SmartHui

## 1. Cài đặt Python
Đảm bảo bạn đã cài đặt Python 3.10 trở lên.

## 2. Cài đặt thư viện
Mở PowerShell tại thư mục này và chạy:
```powershell
pip install -r requirements.txt
```

## 3. Chạy ứng dụng
Sử dụng file script `run.ps1` hoặc chạy lệnh sau:
```powershell
python main.py
```
